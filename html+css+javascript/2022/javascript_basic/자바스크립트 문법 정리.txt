[변수 선언 및 할당]
	-const : 값이 초기화되면 재할당 불가
	const 변수명 = 값
	-let : 값이 초기화된 후 재할당 가능
	let 변수명 = 값
			  
[문자열 템플릿]
	const weight = 100;
	case1)
	console.log('몸무게는' + weight + '입니다.');
	case2)
	console.log(`몸무게는 ${weight}kg 입니다.`);
	
[함수] 
	-함수선언문
	function 함수명(매개변수) {
		실행문;
	}
	-함수표현식 : 선언후 호출가능
	const 험수명 = function(매개변수){
		실행문;
	}
	-화살표함수(람다표현식) : 선언후 호출가능
	 : 주로 고차함수의 파라미터로 함수를 전달할떄 사용.
	const 함수명 = (매개변수) => {
		실행문;
	}
	
[변수]
	전역변수 : 최상위 함수 외부에 선언된 변수명
	지역변수 : 함수 내부에 선언된 변수(매개변수 포함)
	
[전역객체](window == globalThis)
	:브라우저 실행환경에서 미리 생성된 내장객체
	
[배열]	
	-배열 생성
	:const arr = []; //빈배열 생성
	-배열 길이
	배열.length
	-배열의 요소 추가
	배열.push(값)
	-배열관련 API (일반함수,고차함수)
	고차함수란? 함수의 매개값으로 함수를 받는 함수.
	콜백함수란? 고차함수의 매개값으로 반환되는 함수
	
[객체]
	-객체 생성
	const obj = {}; //빈객체 생성
	{프로퍼티 키 : 프로퍼티 값}
	
[이벤트 프로그래밍]
	이벤트 : 사건
	이벤트 소스 : 이벤트가 발생된 요소(element)
	이벤트 객체 : 브라우저가 이벤트 정보를 갖는 객체를 생성
			  이벤트 핸들러의 첫번째 매개변수로 받음	
	이벤트 핸들러(리스너) : 이벤트가 발행했을때 처리할 로직(함수)
	이벤트 타입 : 이벤트 종류(키보드, 마우스, 브라우저)
